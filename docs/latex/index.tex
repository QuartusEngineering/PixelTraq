The Pixel\+Traq library is a C++ library developed to help users integrate cameras calibrated using the Pixel\+Traq \doxylink{classCamera}{Camera} Calibration system into their projects. The library contains ready made camera model implementations and code for common workflows such as image undistortion, 3d reconstruction, and stereo rectification.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{External Links}{External Links}}\label{index_autotoc_md1}

There are a few different ways to include the library in your project. The following sections outline how to integrate the library into your project.\hypertarget{index_autotoc_md3}{}\doxysubsection{\texorpdfstring{Stand Alone to Run Examples}{Stand Alone to Run Examples}}\label{index_autotoc_md3}
If you are only interested in running the examples or tools, you can clone the library directly using\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/QuartusEngineering/PixelTraq.git}

\end{DoxyCode}


The examples can be built as long as you have cmake installed using the following steps\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir\ build}
\DoxyCodeLine{cd\ build}
\DoxyCodeLine{cmake\ ..\ -\/DCMAKE\_BUILD\_TYPE=Release\ -\/DCMAKE\_CXX\_FLAGS="{}-\/O2"{}}
\DoxyCodeLine{cmake\ -\/-\/build\ .\ -\/-\/parallel\ 2\ -\/-\/config\ Release}

\end{DoxyCode}
 Replace --parallel 2 with --parallel \texorpdfstring{$<$}{<}number of cores in your machine\texorpdfstring{$>$}{>}.\hypertarget{index_autotoc_md4}{}\doxysubsection{\texorpdfstring{Add to a Project Using CMake}{Add to a Project Using CMake}}\label{index_autotoc_md4}
\hypertarget{index_autotoc_md5}{}\doxysubsubsection{\texorpdfstring{Add as a Subdirectory}{Add as a Subdirectory}}\label{index_autotoc_md5}
To add Pixel\+Traq as a subdirectory, clone the project into your main project folder or add it as a submodule. 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/QuartusEngineering/PixelTraq.git}

\end{DoxyCode}


Next, add the following line to your project\textquotesingle{}s CMake\+Lists.\+txt file 
\begin{DoxyCode}{0}
\DoxyCodeLine{add\_subdirectory(PixelTraq)}

\end{DoxyCode}


You will also need to target the library to link to your execuatable 
\begin{DoxyCode}{0}
\DoxyCodeLine{add\_executable(target\ main.cpp)}
\DoxyCodeLine{target\_link\_libraries(target\ PUBLIC\ PixelTraq)}

\end{DoxyCode}


Any source files that use the Pixel\+Traq library need to include the entry point header file 
\begin{DoxyCode}{0}
\DoxyCodeLine{\#include\ <PixelTraq/pixeltraq.h>}

\end{DoxyCode}
